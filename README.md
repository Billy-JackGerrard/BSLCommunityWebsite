# BSL Community Website

A community hub for British Sign Language (BSL) events in Edinburgh (and in the future possibly Scotland/UK). The site aggregates socials, classes, and community events into one place so people don't miss what's happening.

## What it does

- Browse upcoming BSL events in list, calendar, or map view
- Filter by date, distance, accessibility features, and more
- Submit events for admin review and approval
- Share events via social links or the native share sheet on mobile
- View event details including location, recurrence, accessibility info, and social media links

## Tech stack

- **Frontend**: React (TypeScript) with Framer Motion for animations
- **Backend**: Supabase (Postgres + Auth + Storage)
- **Maps**: Leaflet
- **Styling**: Plain CSS with CSS custom properties for theming

## Admin features

- Approve/reject submitted events via an admin queue
- Edit event details and manage recurrence
- Manage homepage and about page content
- View and respond to contact messages
